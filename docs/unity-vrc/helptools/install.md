# Installation

To install **LogicCuteGuy Help Tools** into your Unity project, we recommend using the **VRChat Creator Companion (VCC)**. This ensures you always have the latest version and all dependencies are managed automatically.

## Quick Install (VCC)

If you have the VRChat Creator Companion installed, you can click the button below to add our repository and install the package with one click.

## Manual Installation

1. Open your Unity Project.
2. Go to `Window` > `Package Manager`.
3. Click the `+` button in the top left.
4. Select `Add package from git URL...`.
5. Paste the following URL:
   ```text
   https://github.com/LogicCuteGuy/com.logiccuteguy.helptools.git
   ```
6. Click **Add**.

## Requirements
- **Unity 2022.3.x** (VRChat Recommended)
- **VRChat SDK 3.0** (Base requirement for most tools)
