# Help Tools Overview

The **LogicCuteGuy Help Tools** suite is a collection of professional-grade Unity Editor tools discovered in the `Editor` folder. These are designed to streamline scene management, automate optimization, and enhance world-building workflows.

## Feature Categories

- **🚀 Scene Optimization**: Batch process lightmaps, materials, and shaders to keep your project performing at its best.
- **📦 Hierarchy Management**: Smart parenting and sorting tools to keep your project structure clean.
- **🎨 Material Utilities**: Quickly map textures to shaders and replace materials across complex hierarchies.
- **🛠️ World Building**: Randomize placement and rotation to create natural-looking environments fast.

## Installation

For a seamless setup, we recommend using the **VRChat Creator Companion (VCC)**. Click the install button above for a quick one-click setup, or visit our [Detailed Installation Guide](./install.md) for manual steps using Git URLs.

## Getting Started

Once installed, all tools are accessible from the top menu under **LogicCuteGuy > HelpTools**.

Most tools are designed to work on the current selection in the Hierarchy or Project view. Select the objects you want to modify, then run the desired tool from the menu.
