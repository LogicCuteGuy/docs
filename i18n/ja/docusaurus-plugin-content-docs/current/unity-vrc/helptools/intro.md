# Help Tools (ヘルプツール) の概要

**LogicCuteGuy Help Tools** スイートは、`Editor` フォルダにあるプロフェッショナルグレードの Unity エディターツールのコレクションです。これらは、シーン管理の合理化、最適化の自動化、およびワールド構築ワークフローの強化を目的として設計されています。

## 機能カテゴリ

- **🚀 シーンの最適化**: ライトマップ、マテリアル、およびシェーダーをバッチ処理して、プロジェクトのパフォーマンスを最高の状態に保ちます。
- **📦 階層管理**: 階層構造をクリーンに保つためのスマートなペアレンティングおよびソートツール。
- **🎨 マテリアルユーティリティ**: テクスチャをシェーダーに素早くマッピングし、複雑な階層全体でマテリアルを置換します。
- **🛠️ ワールド構築**: 配置と回転をランダム化して、自然な雰囲気の環境を迅速に作成します。

## インストール

シームレスなセットアップのために、**VRChat Creator Companion (VCC)** の使用をお勧めします。上のインストールボタンをクリックしてワンクリックでセットアップするか、Git URL を使用した手動の手順については、[詳細なインストールガイド](./install.md)をご覧ください。

## はじめに

インストールすると、すべてのツールはトップメニューの **LogicCuteGuy > HelpTools** からアクセスできます。

ほとんどのツールは、Hierarchy または Project ビューでの現在の選択物に対して機能するように設計されています。変更したいオブジェクトを選択し、メニューから目的のツールを実行してください。
