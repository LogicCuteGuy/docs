# Random Rotate Children (子のランダム回転)

**メニューパス**: `GameObject > LogicCuteGuy > Random Rotate Children`

繰り返される環境オブジェクトの機械的な見た目を、ローカルの回転にバリエーションを加えることで素早く解消します。

## 主な機能

- **回転モード**:
    - **Offset From Current**: オブジェクトの現在の回転に対して、ランダムな値（±最大値以内）を加えます。既存の配置に「躍動感」を加えるのに最適です。
    - **Absolute Range**: 指定した最小値 (Min) と最大値 (Max) の Vector3 の間で、完全に新しいランダムな回転を生成します。
- **ターゲットフィルタリング**:
    - **Require MeshRenderer**: 目に見えるオブジェクトのみを回転させます。
    - **Recursive**: 子の階層全体を処理するか、直下の子のみを処理するかを選択できます。
- **シーンの統合**: 適切に保存されるように、シーンを自動的に dirty としてマークします。

## 使い方

1. 1 つ以上の **親 (Parent)** GameObject を選択します。
2. **Rotation Mode** を選択します。
3. **Min/Max** または **Max Offset** の値を設定します。
4. **Apply To Selected** をクリックします。

> [!TIP]
> 木や岩などの自然アセットの場合、Y 軸のオフセットを 360 に設定すると、オブジェクトを直立させたまま、すべてのオブジェクトに完全にユニークな向きを与えることができます。
