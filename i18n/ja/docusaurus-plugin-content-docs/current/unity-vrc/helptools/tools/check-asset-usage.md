# Check Asset Usage In Scene (シーン内アセット使用状況確認)

**メニューパス**: `アセット/フォルダを右クリック > Check Usage in Scene`

現在開いているシーンをスキャンして、どのアセットファイルが使用されており、どれが不要（冗長）であるかを特定する高度な分析ツールです。

## 主な機能

- **包括的な検索**: すべての GameObject とすべてのコンポーネントの `SerializedProperty` をスキャンして、選択に一致するオブジェクト参照を探します。
- **階層ツリービュー**: Project ウィンドウを反映した、フォルダベースのツリー構造で結果を表示します。
- **集計統計**: フォルダごとに、総ファイル数、使用済み vs 未使用の数、累計ファイルサイズなどの要約データを表示します。
- **フィルタリングと検索**:
    - **Show Only Unused**: プロジェクト内の不要なアセットを即座に特定します。
    - **Live Filter**: 結果内の特定のファイルを検索します。
- **インタラクティブな結果**: 結果をクリックすると Project ビューでアセットが「ピン（強調表示）」されます。また、使用中のアセットを展開すると、どの GameObject がそれを参照しているかを正確に確認できます。
- **並べ替え可能な列**: 名前、使用状況、またはファイルサイズで並べ替え、最大の最適化効果が得られるアセットを見つけることができます。

> [!TIP]
> VRChat ワールドを公開する前にこのツールを使用して、プロジェクトに含まれているものの、シーン内では実際には何の機能も果たしていない重いテクスチャやメッシュを見つけ出してください。
