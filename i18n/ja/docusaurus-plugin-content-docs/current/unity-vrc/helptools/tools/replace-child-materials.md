# Replace Or Remove Child Materials (子のマテリアルを置換または削除)

**メニューパス**: `GameObject > LogicCuteGuy > Replace Or Remove Child Materials`

階層深くにあるマテリアルの割り当てを一括管理するためのユーティリティです。特定のマテリアルをターゲットにして、別のマテリアルに置き換えるか、そのスロット自体を完全に削除することができます。

## 主な機能

- **置換モード**: `Material A` のすべてのインスタンスを `Material B` に入れ替えます。
- **削除モード**: `Material A` が含まれているマテリアルスロットを完全に削除し、マテリアルリストを縮小します。
- **階層的な深さ**: すべての子オブジェクト、およびすべての MeshRenderer 内のすべてのマテリアルスロットをスキャンします。
- **選択的なクリーンアップ**: 置換先を `None` に設定すると、スロット数はそのままに特定のスロットをクリアできます。

## 使い方

1. 1 つ以上の **親 (Parent)** オブジェクトを選択します。
2. 変更したい **Target Material** を割り当てます。
3. **Action** (Replace または Remove) を選択します。
4. **Apply To Selected** をクリックします。

> [!NOTE]
> このツールはシーンを正しく dirty としてマークし、Undo システムを使用するため、大規模なプロダクションシーンでも安全に使用できます。
