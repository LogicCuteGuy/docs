# ตรวจสอบการใช้งาน Asset ในฉาก (Check Asset Usage In Scene)

**เมนู**: `คลิกขวาที่ Asset/Folder > Check Usage in Scene`

เครื่องมือวิเคราะห์ขั้นสูงที่จะสแกนฉากที่เปิดอยู่ในปัจจุบัน เพื่อระบุว่าไฟล์ Asset ใดที่กำลังถูกใช้งาน และไฟล์ใดที่ซ้ำซ้อนหรือไม่จำเป็น

## ฟีเจอร์

- **การค้นหาที่ครอบคลุม**: สแกนทุก GameObject และทุก `SerializedProperty` ของคอมโพเนนต์เพื่อหาการอ้างอิงออบเจกต์ที่ตรงกับสิ่งที่คุณเลือก
- **มุมมองแบบลำดับขั้น (Tree View)**: แสดงผลในโครงสร้างแบบโฟลเดอร์ เหมือนกับที่คุณเห็นในหน้าต่าง Project
- **สถิติรวม**: โฟลเดอร์จะแสดงข้อมูลสรุป: จำนวนไฟล์ทั้งหมด, จำนวนที่ใช้ vs ไม่ได้ใช้ และขนาดไฟล์รวมทั้งหมด
- **การกรองและการค้นหา**:
    - **Show Only Unused**: ระบุส่วนที่ไม่ได้ใช้งานและถ่วงน้ำหนักโปรเจกต์ของคุณได้อย่างทันที
    - **Live Filter**: ค้นหาไฟล์ที่ต้องการภายในผลลัพธ์
- **ผลลัพธ์ที่โต้ตอบได้**: คลิกที่ผลลัพธ์ใดก็ได้เพื่อ "Ping" หา Asset นั้นในหน้าต่าง Project หรือขยาย Asset ที่ใช้งานอยู่เพื่อดูว่าออบเจกต์ใดในฉากที่อ้างอิงถึงมันบ้าง
- **คอลัมน์ที่จัดเรียงได้**: จัดเรียงตามชื่อ, สถานะการใช้งาน หรือขนาดไฟล์ เพื่อค้นหาโอกาสในการเพิ่มประสิทธิภาพสูงสุด

> [!TIP]
> ใช้เครื่องมือนี้ก่อนการอัปโหลดโลก VRChat เพื่อค้นหา Texture หรือ Mesh ที่มีขนาดใหญ่ซึ่งรวมอยู่ในโปรเจกต์แต่ไม่ได้ถูกใช้งานจริงในฉาก
