# ภาพรวมเครื่องมือช่วยเหลือ (Help Tools Overview)

ชุดเครื่องมือ **LogicCuteGuy Help Tools** คือการรวบรวมเครื่องมือ Unity Editor ระดับมืออาชีพที่พบในโฟลเดอร์ `Editor` เครื่องมือเหล่านี้ออกแบบมาเพื่อเพิ่มความคล่องตัวในการจัดการฉาก ปรับแต่งประสิทธิภาพโดยอัตโนมัติ และเสริมสร้างเวิร์กโฟลว์การสร้างโลกให้ดียิ่งขึ้น

## หมวดหมู่ฟีเจอร์

- **🚀 การปรับแต่งฉาก (Scene Optimization)**: ประมวลผล Lightmap, Material และ Shader เป็นชุดเพื่อให้โปรเจกต์ของคุณทำงานได้ดีที่สุด
- **📦 การจัดการลำดับขั้น (Hierarchy Management)**: เครื่องมือจัดกลุ่มและจัดเรียงอัจฉริยะเพื่อให้โครงสร้างโปรเจกต์ของคุณสะอาดตา
- **🎨 ยูทิลิตี้วัสดุ (Material Utilities)**: จับคู่ Texture กับ Shader อย่างรวดเร็ว และเปลี่ยน Material ในลำดับขั้นที่ซับซ้อน
- **🛠️ การสร้างโลก (World Building)**: สุ่มการวางตำแหน่งและการหมุนเพื่อสร้างสภาพแวดล้อมที่ดูเป็นธรรมชาติได้อย่างรวดเร็ว

## การติดตั้ง

เพื่อการตั้งค่าที่ราบรื่น เราขอแนะนำให้ใช้ **VRChat Creator Companion (VCC)** คลิกปุ่มติดตั้งด้านบนเพื่อตั้งค่าด้วยการคลิกเพียงครั้งเดียว หรือไปที่ [คู่มือการติดตั้งโดยละเอียด](./install.md) สำหรับขั้นตอนการติดตั้งด้วยตนเองโดยใช้ Git URL

## เริ่มต้นใช้งาน

เมื่อติดตั้งแล้ว สามารถเข้าถึงเครื่องมือทั้งหมดได้จากเมนูด้านบนภายใต้ **LogicCuteGuy > HelpTools**

เครื่องมือส่วนใหญ่ถูกออกแบบมาให้ทำงานกับสิ่งที่คุณเลือกในหน้าต่าง Hierarchy หรือ Project เลือกออบเจกต์ที่คุณต้องการแก้ไข จากนั้นเลือกเครื่องมือที่ต้องการจากเมนู
